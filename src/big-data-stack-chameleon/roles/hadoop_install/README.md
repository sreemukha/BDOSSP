# ansible-role-hadoop_install
Ansible role for installing hadoop

# Installation

```bash
$ git submodule add https://github.com/futuresystems/ansible-role-hadoop_install.git roles/hadoop_install
```

# Usage

```yaml
- role: hadoop_install
  hadoop_version: 2.7.1
```
