# ansible-role-hbase_configure
Ansible role for configuring HBase

# Installation

```bash
$ git submodule add https://github.com/futuresystems/ansible-role-hbase_configure.git roles/hbase_configure
```

# Usage

```yaml
- role: hbase_configure
```
